# AGENTS

This file defines the shared operating model for coding agents working in this repository.

## Purpose

- Keep automation predictable and reviewable.
- Make project setup and implementation steps explicit.
- Ensure plan-driven execution with a single source of truth in `PLAN.md`.

## Working agreement

1. Read `PLAN.md` before starting implementation work.
2. Update `PLAN.md` as tasks are started/completed.
3. Prefer small, verifiable changes over large rewrites.
4. Keep docs synchronized with code and structure changes.
5. Do not remove existing user changes unless explicitly requested.

## File roles

- `AGENTS.md`: General cross-agent rules and expectations.
- `CLAUDE.md`: Claude-specific notes and execution style.
- `PLAN.md`: Current plan, milestones, and progress tracking.

## Pull request checklist

- Scope is aligned with `PLAN.md`.
- New files/folders are documented in `README.md`.
- Basic verification steps are recorded.
- Follow-up tasks are listed if anything is intentionally deferred.
